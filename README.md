# Dashboard de Vendas - XBOX GAME PASS SUBSCRIPTIONS SALES

O dashboard de vendas do XBOX é um projeto da trilha de aprendizagem do Bootcamp Santander 2025 - Ciência de Dados com Python.
Para criar o dashboard recebemos um arquivo base com algumas informações, incluindo a base de dados em si, onde consta a relação de vendas de assinaturas do XBOX Game Pass e outras assinaturas adicionais.
No meu dashboard aproveitei algumas informações desenvolvidas durante as aulas e acrescentei novas análises: Quantidade de Inscrições por mês, Descontos concedidos de acordo com o plano e a quantidade de assinaturas com ou sem auto renovação.
Com esses itens é possível identificar meses que tiveram uma melhor performance nas vendas, entender quais planos carecem de mais descontos ou não e também qual o tipo de plano que possui mais usuários sem auto-renovação.

Para a reprodução e visualização do dashboard é necessário apenas realizar o download do arquivo.
